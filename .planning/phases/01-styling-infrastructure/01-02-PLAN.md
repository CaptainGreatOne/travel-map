  <files>src/components/Sidebar.jsx</files>

    Replace all inline style={{...}} with Tailwind classes.

    Key mappings for this component:
    - width: '280px' → w-[280px] or w-72
    - backgroundColor: '#2d4356' → bg-secondary
    - color: '#f5f5f5' → text-gray-100
    - color: '#f4a261' → text-primary

    - gap: '0.5rem' → gap-2
    - borderRadius: '8px' → rounded-lg
    - transition → transition-colors duration-200

    Remove onMouseEnter/onMouseLeave hover handlers - use Tailwind hover: prefix instead.
